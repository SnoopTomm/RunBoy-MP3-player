# RunBoy-MP3-player
@ walkman inspired mp3 player that runs on linux using winmap themes built using a raspberry pie zero 2

RunBoy is a walkman inspired mp3 player that runs on a raspberry pi zero 2 using QMMP on the Linux opperating system
It includes an LCD display, HiFiBerry DAC audio, 32gb of storage, and a DC lithium battery as the power suply.
Further details and renders will be included as project is further developed
